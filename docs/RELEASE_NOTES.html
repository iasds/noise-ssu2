RELEASE NOTES - go-i2p/go-noise

Release notes for: go-noise Version 0.1.55

This release is recommended only for developers at this time. The API is likely to be unstable. The first general-purpose-release will be version 0.2.0.

This release features newly-integrated SSU2 libraries. Ratchet(ECIES-X25519-AEAD) encryption has been factored out of go-i2p and moved into go-noise. Numerous bugs have been fixed after observation on the live network.
